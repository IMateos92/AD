# Cómo usar

## Ejemplo básico

```python
from src import example

# Tu código aquí
result = example.hello("mundo")
print(result)
```

## Configuración

Puedes configurar la aplicación mediante variables de entorno o archivos de configuración.

```bash
export CONFIG_VAR=value
python -m src
```

## Comandos disponibles

### Ver ayuda

```bash
python -m src --help
```

### Ejecutar con opciones

```bash
python -m src --option value
```
